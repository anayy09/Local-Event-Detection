About This Project

This system automatically detects, clusters, and summarizes local news events from various online sources.

How It Works

  1. 1. News Scraping
    Articles are collected from configured local news websites and RSS feeds. Duplicate articles are skipped.
  2. 2. Preprocessing
    The raw text content of each article is cleaned (HTML removed, whitespace normalized) and prepared for analysis.
  3. 3. Named Entity Recognition (NER)
    Key entities (like People, Organizations, Locations, Dates) are identified within the article text using NLP models.
  4. 4. Embedding & Clustering
    Articles are converted into numerical representations (embeddings) based on their content. Articles with similar content are grouped together into clusters using algorithms like K-Means. Each cluster represents a potential news event.
  5. 5. Topic Modeling
    For each cluster, topic modeling techniques (like BERTopic) are used to identify the main keywords and assign a representative topic name.
  6. 6. Summarization
    An abstractive or extractive summary is generated for each event cluster, condensing the information from all articles within that cluster.
  7. 7. Analysis & Visualization
    Statistics and visualizations are created based on the collected data, such as entity frequency and cluster distribution.
  8. 8. Web Interface
    A web application (built with Flask) allows users to browse the detected event clusters, view summaries, explore related articles, and see analytics.
  9. 9. Automation
    The entire pipeline (Steps 1-7) is scheduled to run automatically at regular intervals (e.g., every 6 hours) to keep the event data up-to-date.

Technology Stack

This project utilizes Python and several libraries, including: Flask (Web Framework), SQLAlchemy (Database ORM), Requests/BeautifulSoup/Newspaper3k/Feedparser (Web Scraping), NLTK/spaCy (Preprocessing, NER), Scikit-learn/Sentence-Transformers (Embeddings, Clustering), BERTopic (Topic Modeling), Transformers/Torch (Summarization Models), Matplotlib/Seaborn (Visualization), APScheduler (Task Scheduling), and Bootstrap (Frontend Styling).
